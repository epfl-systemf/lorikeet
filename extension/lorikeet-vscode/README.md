# lorikeet-vscode

Syntax highlighting for Lorikeet configuration files in VSCode (`.lorikeet.conf` files).

## Features

This extension provides syntax highlighting for Lorikeet configuration files,using HOCON highlighting as a base and embedded Scala syntax highlighting inside triple-quoted strings for pattern and rewrite templates. It also includes additional highlighting for Lorikeet specific constructs like metavariables and pattern operators.

## Requirements

For this extension to fully work, make sure you have the [Scala Syntax (official)](https://marketplace.visualstudio.com/items?itemName=scala-lang.scala) extension installed.
